Why Now?

01 / Background

📍 항공 발권 대리점 승격에 따른 과제

올해 Fly Unity의 항공 발권 대리점 승격으로 인해, 안정적인 발권 볼륨(Volume) 확보가 최우선 과제가 되었습니다. 기존 B2B 채널만으로는 급변하는 항공 시장의 요구 수량을 맞추기에 한계가 존재할 수 있습니다.

📍 랜드사의 딜레마: B2C 직접 진출의 리스크

수익성을 위해 직판(B2C)이 필요하지만, 이를 위해 별도의 CS 인력, 마케팅 부서, 웹 시스템을 구축하는 것은 고비용/고위험 투자입니다. 기존 소매 거래처와의 마찰 또한 우려되는 부분입니다.

"별도의 투자 없이, 즉각적으로
항공+지상비 상품을 판매할 디지털 창구가 필요합니다."

03 / Tech Edge
Preview Image

스마트 가이드북 (Smart Guidebook)

기존의 종이 일정표 대신, URL 하나로 제공되는 모바일 웹 일정표입니다. 고객은 앱 설치 없이 일정 확인, 현지 정보 습득, 추가 옵션 결제까지 가능합니다.

  • ✓ 현지 도착 후 즉시 사용 가능한 정보 제공
  • ✓ 투어메이커의 자동화된 기술로 제작 시간 단축
  • ✓ Fly Unity 상품의 매력을 디지털로 시각화

온라인 자동 견적 & 모객 시스템

소매 여행사의 복잡한 상담 과정을 디지털화합니다. 고객이 온라인에서 직접 견적을 요청하면, 양사 담당자가 시스템을 통해 신속하게 공동 대응합니다.

  • 단순 문의 응대 최소화 (인건비 절감) ✓
  • 24시간 접수 및 데이터 기반 세일즈 ✓
  • Fly Unity의 항공 좌석 실시간 소진 지원 ✓
